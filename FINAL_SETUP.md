## 🚀 Deployment

### Option 1: Static Hosting (Recommended)

**Netlify / Vercel / Firebase:**
```bash
# Build
npm run build

# Deploy folder: dist/
```

**Drag & drop:** Upload folder `dist/` ke hosting

### Option 2: Web Server

**Apache / Nginx:**
```bash
# Copy dist/ to web root
cp -r dist/* /var/www/html/

# Or Windows
xcopy dist\* C:\inetpub\wwwroot\ /E /I
```
