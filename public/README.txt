LOGO KEMENKES
=============

Silakan tambahkan file logo Kemenkes dengan nama:
- kemenkes-logo.png

File ini akan muncul di header aplikasi.

Jika file tidak ada, aplikasi akan menggunakan logo SVG fallback.

Ukuran rekomendasi:
- Width: 200-400px
- Height: 200-400px
- Format: PNG dengan background transparan
- File size: < 100KB
